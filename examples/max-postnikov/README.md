# Example: Max Postnikov's Human Core

This is a real Human Core — the protocol author's own profile — **flattened for publication**. The live version ([humancontext.pro/p/postnikov](https://humancontext.pro/p/postnikov)) is longer, more candid, and partly private.

What was changed for this public example:

- **Names and places of third parties removed.** `PEOPLE.md` describes roles, never identities.
- **High-sensitivity files are deliberately generic.** Real `PATTERNS.md` and `BOUNDARIES.md` contain material you share with agents you control, not with the internet. Here they show the *structure* with softened content.
- **Numbers removed.** Real `GOALS.md` has concrete revenue and audience targets; an example doesn't need them.
- **Translated to English and shortened.** The original is in Russian — write yours in whatever language you think in.

A few `[? VERIFY]` markers are left in on purpose: that's how unconfirmed, AI-inferred content looks in a living profile.

```
core/       WHO.md  VOICE.md  NOW.md  GOALS.md
identity/   INFLUENCES.md  INTERESTS.md  PEOPLE.md
deep/       PATTERNS.md  BOUNDARIES.md
META.md
```
